# dotfiles

My personal dotfiles.

## Required packages

- stow
- zsh
- zsh-completions
- zsh-autosuggestions
- zsh-syntax-highlighting
- fzf
- oh-my-zsh-git
- (more to follow)

## Installation

There are currently two targets: `headless` and `desktop`.

### raptor

```bash
make [target]
```

## Uninstall

```bash
make uninstall-[target]
```
