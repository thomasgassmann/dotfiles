# dotfiles

My personal dotfiles.

## Required packages

- zsh
- zsh-completions
- zsh-autosuggestions
- zsh-syntax-highlighting
- fzf
- (more to follow)

## Installation

There are currently three targets: `raptor`, `nas` and `matebook`.

### raptor

```bash
make [target]
```

## Uninstall

```bash
make uninstall-[target]
```
